Installation
============

Requirements
------------

* Python >= 3.7
* pandas
* protobuf
* rdkit
* ord_schema

Install from PyPI
-----------------

.. code-block:: bash

   pip install ord-rxn-converter

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/cwru-sdle/ord_rxn_converter.git
   cd ord_rxn_converter
   pip install -e .

Development Installation
------------------------

.. code-block:: bash

   git clone https://github.com/cwru-sdle/ord_rxn_converter.git
   cd ord_rxn_converter
   pip install -e ".[dev,docs]"
